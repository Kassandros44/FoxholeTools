# FoxholeTools

A Self contained tool package for the game Foxhole: Persistent World Warfare. <br/>
This application contains mapping and intel tools, along with stockpiling and logistics tools. Incorperating a Discord login for user profiles and permissions.<br/>
<br/>
**Currently Included Features**
- Stockpiling tool, traking of multiple stockpiles
- Logging of user inputted stockpile entrys and changes
- Quotas for stockpile management
- Colour coded values for 'at a glance' identifing of quota levels
- Foxhole world map view
- Map markers for towns and infrastructure
